# UltraPredict Automation

## Installation
1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Run automatic fetching:
   ```sh
   python fetch_data.py
   python predict.py
   python database.py
   ```

## Automation
- Use `run_fetch.sh` for Linux/macOS.
- Use `run_fetch.bat` for Windows.
